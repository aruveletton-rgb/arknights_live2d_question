# 合规与敏感信息边界

## 1. 非官方性质

本项目是非官方同人计划，不隶属于、不代表、不声称获得《明日方舟》、鹰角网络、Yostar 或其他相关官方实体授权。

## 2. 不得上传到本说明仓库

- 官方游戏拆包资源、立绘、贴图、Live2D 或音频。
- 未授权角色语音、声优克隆音色或来源不明的声音模型。
- 任何项目源代码、模型文件、图片、声音、字体或安装包。
- 真实 `.env`、API Key、令牌、私钥、服务器凭据或订阅链接。
- 包含个人隐私、完整用户语音或认证信息的日志。

## 3. 可以记录的素材说明

本仓库可以用文字记录计划使用的素材，但不能上传素材本身。记录至少包含：

| 项目 | 需要说明的内容 |
| --- | --- |
| 名称 | 素材或音色的公开名称 |
| 来源 | 作者和原始公开链接 |
| 许可 | 允许使用、修改和公开分发的依据 |
| 用途 | 计划用于模型、背景、声音或其他位置 |
| 状态 | 待审核、可使用、不可使用或已移除 |

授权证明包含个人信息时，只记录由仓库拥有者安全保存证明的位置，不把证明原件放入公开仓库。

## 4. 发布前需要确认

- 项目代码许可证由实现仓库明确。
- 上游项目许可证和修改说明被保留。
- 模型、图片、声音、背景和字体都有来源记录。
- README 明确说明非官方性质。
- 商业直播、广告、会员和周边用途已经重新评估授权风险。
